# ログインAPI

- **URL**: `/api/login_servlet/login`
- **Method**: `POST`
- **Request Parameter**:


| 필드명     | 타입    | 데이터사이즈  |  설명         |
|:----------|:-------|:------------|:-------------|
| email     | String | 30          | 비밀번호     |
| password  | String | 30          | 비밀번호     |


### ✅ 성공 응답 필드

| 필드명     | 타입    | 설명         |
|:----------|:-------|:-------------|
| authority | String | 유저 권한 (admin/user) |
| userId    | String | 작성자 ID    |


### ❌ 오류 코드

| 코드       | 에러 메세지                    |
|:----------|:------------------------------|
| 200系以外  | ログイン情報を確認してください。|
